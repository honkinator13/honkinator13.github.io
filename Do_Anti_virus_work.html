How does Anti-Virus Work?

How they protect you?

Anti-Virus software protects you by scanning your files and programs for known malware signatures. They are able to spot it by having a database where they can identify the virus. When it finds a match, it can quarantine or delete the infected file to prevent harm to your system.

Do You Need Antivirus Software?

You can use the base anti-virus in your operating system as they are able to cover the most of the services of a paid anti-virus. However these base anti-virus are not able to have the same features as paid anti-virus. This can mean that if you work in a field that requires it, like a office job, it might require these anti-virus solutions.
