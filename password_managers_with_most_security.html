What makes a password manager secure?

Password managers are secure by storing your password in an encrypted vault, it is only accessible to the one with the password which would be the user. It is extremely difficult to access the password and it is also secure through multiple factor authentication.

RoboForm

RoboForm is a very secure password manager as they have multiple factor authentication to access the passwords. It also has a zero knowledge architecture. It's encryption is very strong which can deter hackers from infiltrating the account.
